# Sistem Manajemen Kontak
<p align="justify">
Sistem ini digunakan untuk mengelola data kontak secara praktis. Di dalamnya tersedia fitur untuk menambah kontak baru, menampilkan daftar kontak, 
mengedit data yang sudah ada, dan menghapus kontak. Seluruh data disimpan sementara melalui session sehingga tidak memerlukan database.
</p>

## Fitur Utama
* Form tambah kontak dengan validasi
* Daftar kontak dalam tampilan tabel
* Fitur edit kontak
* Fitur hapus kontak
* Penyimpanan data menggunakan session

## Cara Menjalankan
1. Install dan jalankan XAMPP
2. Letakkan folder project ke dalam:
   ```
   htdocs/manajemen-kontak/  (XAMPP)
   ```
4. Jalankan Apache melalui XAMPP Control Panel
5. Buka browser dan akses:
    ```
    http://localhost/manajemen-kontak/ 
    ```
